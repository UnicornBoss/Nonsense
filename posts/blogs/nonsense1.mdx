# 开发

通过**阮一峰**老师搭建的[网道](https://wangdoc.com/)课程，从零开始学习了Javascript，发现这门语言还是很深奥的，同时也发现自己之前的根基非常不牢靠，平时的知识都是面向需求学习的，导致学下来的东西东一块，西一块，非常零碎，不成体系。

这一次学习，终于有了一种入门的感觉，也明白了那句话。

> 进步慢一点没关系的，只要每天都在进步就行。

同时最近Javascript 和 React 交叉学习的，这个时候才体会到什么是真正的大脑休息（有科学依据说，休息并不是睡觉，而是换着做事），看A累了就看会B，看B累了就再看会A，好不快哉。

下面这个代码片段，是我最近印象最深的片段:

```javascript
(function(){ /* code */ }());
// 或者
(function(){ /* code */ })();
```
👆上面两种叫做“立即调用的函数表达式”（Immediately-Invoked Function Expression），简称 IIFE。
